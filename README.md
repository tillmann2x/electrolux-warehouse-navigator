# Electrolux Warehouse Navigator 📦

Um navegador inteligente para gestão e visualização de armazéns, desenvolvido para otimizar processos logísticos da Electrolux.

## 🚀 Funcionalidades

* **Visualização em Tempo Real:** Interface interativa para navegação no estoque.
* **Integração de Dados:** Processamento de fluxos de entrada e saída (SAP).
* **Interface Responsiva:** Otimizado para uso em tablets e desktops.
* **Gestão de Inventário:** Controle simplificado de posições e itens.

## 🛠️ Tecnologias Utilizadas

* **React + Vite** (Frontend rápido e moderno)
* **TypeScript** (Segurança e tipagem de código)
* **Tailwind CSS** (Estilização avançada)
* **Lucide React** (Ícones dinâmicos)
* **Shadcn/UI** (Componentes de interface profissional)
